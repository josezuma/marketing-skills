# Content Brief Generator

Generates research-backed content briefs optimized for AI-search (GEO) visibility.

## When to Use

- Planning a new blog post or article
- Creating documentation that needs to be AI-search visible
- Briefing writers on GEO-optimized content requirements

## Output

A structured brief containing:
1. **Target query cluster** — 5-10 related search queries
2. **GEO requirements** — schema types, citability signals, llms.txt readiness
3. **Key sections** — H2/H3 structure optimized for AI extraction
4. **Competitor analysis** — what existing content covers and gaps to exploit
5. **Statistics needed** — data points that improve citation likelihood

## Example

```
Content Brief: "How to Optimize for AI Search"

Query Cluster: GEO guide, AI search optimization, LLM citation tips
Schema: Article + FAQ
Key Points: 3 statistics needed, compare with competitor content
Sections: What is GEO → Why it matters → Implementation → Tools → FAQ
```
